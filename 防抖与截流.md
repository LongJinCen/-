## 节流

在指定的时间间隔内只会执行一次任务, 即设定一个标识，在指定的时间内，根据这个标识来判断是否可以执行代码

下面的的函数用来让一个函数具有节流的能力

```
function throttle (fn, interval = 300) {
    var canRun = true
    return function () {
        if (!canRun) return 
        canRun = false
        setTimeout(() => {
            fn.call(this, arguments)
            canRun = true
        }, interval)
    }
}
```
